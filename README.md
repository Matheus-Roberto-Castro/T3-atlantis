# Atlantis - Atividade 3

## Tecnologias Utilizadas

- Node.js

- TypeScript


---

## Como Executar o Projeto

### Pré-requisitos

- Node.js v22.14.0 instalado

- NPM instalado

## Instalação das Dependências

No diretório do projeto, execute:
```bash
npm install
```

Compilando o Projeto:
```bash
npx tsc
```
Isso irá gerar os arquivos JavaScript dentro da pasta configurada (ex.: js/).

## Executando o Projeto

Após compilar, execute o arquivo desejado com:
```bash
node src/js/app/app.js
```
Ajuste o caminho caso seus arquivos compilados estejam em outra pasta.
